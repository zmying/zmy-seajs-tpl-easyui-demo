# zmy-seajs-tpl-easyui-demo<br>

此项目是对<a href="https://github.com/zmying/zmy-seajs-tpl-demo" title="https://github.com/zmying/zmy-seajs-tpl-demo">zmy-seajs-tpl-demo项目</a>的升再级版，不了解seajs的使用可以查看<a href="https://github.com/zmying/zmy-seajs-demo" title="https://github.com/zmying/zmy-seajs-demo">zmy-seajs-demo项目</a>和<a href="https://github.com/zmying/zmy-seajs-tpl-demo" title="https://github.com/zmying/zmy-seajs-tpl-demo">zmy-seajs-tpl-demo项目</a>,
项目在加入了腾讯arttemplate模板引擎的整合基础上，加入了easyui实现了easyui组件嵌入seajs模块化的实现，<br>
实现数据，页面，的重复利用，如：对easyui的常用表单控件、数据表datagrid、树tree、My97DataPicker时间控件、<br>
editor编辑器等等常用组件动态配置，实现所有表单的json配置，
同时对artTemplate整合，借此了解和加深一下cmd的基本规范。
